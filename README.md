# AI Quiz Builder

> Live demo: _coming when Phase 8 ships_

Multiple-choice quiz generator. Pick a topic, get five questions grounded in Wikipedia, generated by Claude.

This README is a placeholder. Final structure (per `decisions.md` Q22) lands in Phase 8.

## Setup (local)

```bash
cp .env.example .env
# fill in ANTHROPIC_API_KEY, UPSTASH_REDIS_REST_URL, UPSTASH_REDIS_REST_TOKEN
npm install
npm run dev
```
